# API Data Fetcher

## Objective

Fetch data from a public API using Python and display user information.

## Technologies Used

* Python
* Requests Library
* JSON Data Handling

## Features

* Fetches data from API
* Parses JSON response
* Displays user details
* Search user by name
* Exception handling included

## API Used

https://jsonplaceholder.typicode.com/users

## Output

Displays ID, Name and Email of users fetched from API.
